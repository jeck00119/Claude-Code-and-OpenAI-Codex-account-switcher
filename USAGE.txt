===============================================
   ACCOUNT SWITCHER - QUICK START GUIDE
===============================================

HOW TO LAUNCH:
--------------
1. Double-click "start.bat"
   (Shows terminal window)

   OR

2. Double-click "start-hidden.vbs"
   (Cleaner - no terminal window)


FIRST TIME SETUP:
-----------------
1. Log into Claude Code or OpenAI Codex normally (through browser)
2. Launch Account Switcher
3. Click "Save" and give your account a name (e.g., "Work")
4. Repeat for each account you want to save


SWITCHING ACCOUNTS:
-------------------
1. Launch Account Switcher
2. Click "Switch" on the account you want
3. Restart Claude Code or OpenAI Codex
4. Done! You're now using that account (no re-login needed!)


BENEFITS:
---------
- Save multiple accounts once
- Switch instantly without re-authenticating
- No more logging in and out repeatedly
- All data stored locally on your computer
